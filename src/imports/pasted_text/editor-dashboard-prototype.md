Create a modern, professional, high-fidelity GUI prototype for **GUI-DASH02 – Editor Dashboard** of a web-based **Manuscript Management System**.

The dashboard should function as the central operational hub for Editors to monitor manuscript workflows, reviewer assignments, workflow bottlenecks, timeline predictions, revisions, and real-time publishing activities.

The design must prioritize:

* workflow visibility
* operational efficiency
* real-time monitoring
* intelligent reviewer management
* predictive workflow analytics

Use a clean academic publishing SaaS-style interface inspired by:

* Elsevier Editorial Manager
* Springer submission systems
* Jira dashboards
* Linear App
* Notion dashboards
* Monday.com workflow interfaces

---

# DESIGN STYLE

Use:

* modern dashboard layout
* soft blue, white, and gray palette
* clean typography
* glassmorphism or soft shadows
* rounded cards
* responsive desktop-first layout
* professional academic aesthetic
* organized workflow visualization
* minimal clutter
* data-focused UI
